# Irish Hospital Waiting List Dashboard

## Overview
This project analyses Irish hospital outpatient waiting list data using Power BI.  
It explores trends in waiting lists, hospital pressure, patient types, and waiting time distributions.

The aim is to demonstrate healthcare analytics capability using real-world public data.

---

## Tools Used
- Power BI
- Data transformation (Power Query)
- Excel / CSV dataset

---

## Key Insights
- Waiting lists show a sustained upward trend over time  
- Adult patients account for the majority of demand  
- Galway University Hospital has the highest patient load  
- Long waiting times (18+ months) are concentrated in a small number of hospitals  
- Most patients fall into the 0–6 month category, but long waits remain significant
